### Claude Desktop에서 사용 방법

`claude_desktop_config.json` 파일에 다음 내용을 추가하세요:

```json
{
  "mcpServers": {
    "[서비스 식별자]": {
      "command": "docker",
      "args": [
        "run",
        "-i",
        "--rm",
        "-e",
        "[환경변수명]",
        "-e",
        "[이미지 이름]"
      ],
      "env": {
        "[환경변수명]": "[환경변수값]"
      }
    }
  }
}
```
